# Elaris

Live site: https://elaris.framer.media/

## About

Elaris is a front-end experience site built with Vite. It focuses on smooth motion, rich visuals, and a clean content layout.

## Tech Stack

- Vite
- HTML, CSS, JavaScript

## Run Locally

From the project root, move into the Vite app and install dependencies:

```bash
cd vite-project
npm install
```

Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
